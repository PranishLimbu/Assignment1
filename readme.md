# HTML Basics Assignment

## Description
This project is part of an assignment to practice core HTML concepts.  
It demonstrates the use of:
- HTML forms
- HTML tables
- HTML5 semantic elements

The goal of this assignment is to understand proper structure, semantics, and basic webpage organization using HTML.

## Features
- User input form with labels (text input, radio buttons, checkboxes, textarea)
- Organized table with headings, borders, and merged cells
- Webpage structured using HTML5 semantic elements like header, nav, article, aside, and footer

## Technologies Used
- HTML5
- CSS (basic styling)

## Steps to Run the Project
1. Clone the repository or download the ZIP file.
2. Open the project folder.
3. Open the `.html` file in any modern web browser (Chrome, Edge, Firefox).
4. The webpage will load and display all required components.
